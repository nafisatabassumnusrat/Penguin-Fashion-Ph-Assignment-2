 # 🐧 Penguin Fashion — Modern E-Commerce Landing Page

✨ A stylish and responsive fashion landing page built with modern UI/UX principles using Tailwind CSS.

This project showcases a clean, minimal, and visually appealing **winter fashion e-commerce design**, focusing on user experience, layout structure, and responsive design.

---

## 🚀 Live Preview

> 🌐 Fully responsive & optimized for all devices

---

## 🌟 Features

🎨 **Modern UI Design**
- Clean and minimal layout
- Soft color palette (green, yellow, neutral tones)
- Professional typography (Bebas Neue + Roboto)

📱 **Fully Responsive**
- Mobile-first design
- Optimized for tablet & desktop

🛍️ **Product Showcase**
- Women & Men jacket sections
- Product cards with pricing & CTA

⚡ **Interactive Elements**
- Hover effects
- Smooth transitions
- Button animations

🧩 **Component-Based Layout**
- Navbar
- Hero section
- Product grid
- Features section

---

## 🛠️ Tech Stack

- **HTML5**
- **Tailwind CSS**
- **Google Fonts**
- Fully static — no frameworks ⚡
